entry.m script is a collection of the experiments conducted during the project. all functions are called in this script. please open this script if you know little about this project's structure.

functions with name Forming*.m are used to generate constant values.

functions with name Precalc*.m are used to pre-calculate some values to save time.

functions LineGS*.m and PointGS.m are used to conduct G-S iterations.

function V-Cycle.m conducts the V-Cycle preconditioning.

function GaussElim.m conducts the experiments using gauss elimination.

function PCG.m conducts the experiments using preconditioning conjugate gradient method.

function ShowResults.m print the experiment results on the screen according to the input.
